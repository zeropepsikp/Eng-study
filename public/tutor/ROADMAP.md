# AI Tutor — Roadmap

Stage 1 (지금 구현) 이후에 올릴 기능들. 사용자와 함께 단계적으로 고도화한다.

## Stage 1 (DONE — 현재 구현)
- Tutor 패널 (번역 메뉴 오른쪽), 빈 배경
- HF 이미지 3장 생성 → 1개 선택 (1회성, 로컬 저장)
- Gemini 2.5 Flash 대화 (`/api/tutor/chat`, JSON 출력: 교정 + 답변 + 한/영 + 힌트)
- Gemini 2.5 Flash TTS 튜터 음성 (`/api/tutor/tts`), 옵션에서 여성 음성 선택
- 입력: 텍스트 + 음성(브라우저 STT) + Submit
- 상단 버튼 3개: 영어 자막 ON/OFF(기본 OFF) · 상황 초기화(확인 2단계) · 대화 저장
- "?" 아이콘 → 직전 튜터 대사 영어 자막
- 시나리오/페르소나/학습자 마크다운 관리체계 (`tutor/…`)
- 옵션: 튜터 대화 내역(한/영) 보기

## Stage 2 — 비주얼
- Kling 이미지→영상 (2–3초 말하는 모션 루프), 실패 시 정적 이미지 폴백 (`KLING_API_KEY`)
- 상황별 배경 이미지 슬롯 채우기
- 말할 때 입모양/미세 모션, 듣는 중 표시

## Stage 3 — 학습 효과
- 턴 종료 시 "오늘 배운 표현 3개" 요약 카드 → 청크/오늘의 학습에 자동 적재
- 발음/유창성 점수 (STT 결과 vs 목표 문장, WPM·머뭇거림)
- 세션 리포트(사용한 표현·실수·새 어휘·다음 연습거리) → 통계 누적

## Stage 4 — 몰입·동기
- 난이도 자동 조절 (정답률/유창성 기반)
- 시나리오별 미션·성취 게이지(예: "환불 받아내기" 성공/실패 피드백)
- 막힘 도우미: 침묵/머뭇거림 감지 → 한국어 힌트나 보기 제시
- 교정 강도 선택 (즉시 리캐스트 vs 턴 끝 모아서)
- 비용 가드(요청당 글자수·일일 횟수 상한), 키 없을 때 데모 모드

## 관리 메모
- 시나리오 추가: `tutor/scenarios/<id>.md` 만들고 `index.json`에 등록.
- 페르소나/말투: `tutor/persona.md`. 학습자 정보: `tutor/learner.md`.
- 프론트매터는 `title`, `opening`, `difficulty`만 사용(스칼라). 나머지는 본문 자유 서술.
